＃DOM操作（Document Object Model）
DOM操作は、JavaScriptを使用してHTML文書の要素にアクセスし、変更するための方法です。
document.getElementById('open')やdocument.getElementById('modal')などを使用して要素を選択します。

＃イベントリスナー（Event Listener）
イベントリスナーは、特定のイベント（クリックなど）が発生したときに実行される関数です。
open.addEventListener('click', () => {...})やclose.addEventListener('click', () => {...})で、クリックイベントが発生したときにモーダルウィンドウを表示または非表示にする関数を登録しています。

＃CSSのスタイル設定
CSSを使用して要素のスタイルを設定します。
#openや#modal、#maskなどのセレクタを使用して、幅や高さ、背景色、位置などのスタイルが指定されています。

これらの要素を使用して、モーダルウィンドウを作成しました。
詳細を見るボタンをクリックすると、モーダルウィンドウが表示され、閉じるボタンまたは背景のマスクをクリックすると非表示になります。モーダルウィンドウの表示と非表示は、CSSのhiddenクラスを追加または削除することで制御されます。

モーダルウィンドウは、Webページ上にコンテンツを重ねて表示するための効果的な方法です。重要な情報や対話的なコンテンツを表示するために使用され、ユーザーの注意を引き付ける効果があります。
