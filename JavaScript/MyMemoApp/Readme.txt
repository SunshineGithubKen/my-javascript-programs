このシンプルなメモ帳アプリでは、テキストエリアにメモを入力し、保存ボタンでローカルストレージに保存できます。削除ボタンでメモを削除することもできます。メッセージが表示され、保存・削除の結果を確認できます。

ブラウザで表示するメモ帳アプリを作成。
このアプリでは、localStorage を使ってデータを保存・取得・
削除を行う機能を実装しました。
